# EReminisce
Addon for Evernote. Provides reminder according to setted tags.

Tag syntax:
```
ER N
```
Where N - **gap** (in days) beetween reminders day that will increace every reminde event.

Example:
```
ER 1
```
Will remind you after tag creation on days: 1 (N * 0), 3 (N * 1), 6 (N * 2) and so on
